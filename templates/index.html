策略说明

规则来源 / 权重 / 风险限制 / 验证方式

推荐默认策略

优先使用“强共识 + Agent 风控优先策略”：先看市场状态，再筛选 2 个以上策略共同入选、质量分较高、风险分较低、Agent 委员会未风控否决的股票。动作过滤建议选“优先跟踪”，排序选“风险最低”或“质量优先”。

TradingAgents 映射

参考 TauricResearch/TradingAgents 的分析师、牛熊研究、交易员、风控和组合经理流程，但不复制其代码、不默认接入 LLM。当前用本地量价、舆情、市场状态和风险字段生成 Agent 委员会分数与最终动作。

策略来源

当前策略是本地启发式量化规则，不是训练好的收益预测模型。它来自常见短线量价逻辑：成交额、换手、量比、当日涨幅、趋势位置、执行性和风险扣分。

14:30 明日潜力

候选池按成交额优先，剔除 ST、退市、低成交额、涨停和近涨停。综合分约按流动性 30%、动能 28%、趋势 20%、执行性 22% 计算，并对高涨幅、高量比、高换手、高振幅、年内涨幅过大扣分。

科技潜力

先匹配 AI/算力、半导体、机器人、低空航天、智能汽车、新材料、医疗科技等方向，再偏好尚未明显透支的个股。60日和年内涨幅过高会被过滤或扣分。

波段与中长期

波段策略关注 5/10/20 日动量、均线位置和温和放量；中长期策略当前是技术趋势版，关注 60日/年内趋势、波动、流动性和涨幅透支，不包含财务、估值和业绩预测。

当前限制

免费行情字段不稳定，行业/概念识别主要依赖名称和可用字段，存在误判。策略不会保证盈利，必须看“策略验证”里的真实命中率和收益后再决定是否保留。

舆情详情
