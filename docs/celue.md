# 评分荐股收益复核与优化计划

  ## 结论

  保持当前生产策略和安全规则不变，先建立可验证的收益基线，再以影子策略验证改动，禁止直接调权或放宽硬过滤。

  ## 实施重点

  - 补齐 v20 点时决策轨迹、候选召回率、因子单调性、local-only/hybrid 增量和净收益结算。
  - 影子验证四项变体：
      - 回踩/突破分支独立计算，并改为连续形态分；
      - 缺失值按覆盖率向 50 分收缩；
      - 候选增加“最终分乐观上界”召回保护；
      - 高热度改为“热度 + 弱结构”组合观察规则。

  - 风险硬过滤、公司风险核验、板块可靠度、DeepSeek 权限边界、冻结规则不放宽。
  - 在样本充分后再校准动作阈值、TopK、行业上限和板块权重；不做在线自动调权。

  ## 晋级门槛

  每个策略至少满足：

  - 60 个有效交易日；
  - 300 条有效配对样本；
  - 扣除费用和滑点后的净超额收益差，95% 置信下界大于 0；
  - 严重回撤率不高于当前基线；
  - 最终 TopK 候选召回率不低于 99%；
  - 20bp 主口径及 50bp/100bp 压力测试均通过。

  未达到门槛时继续使用当前生产策略，影子结果只能用于研究展示。
